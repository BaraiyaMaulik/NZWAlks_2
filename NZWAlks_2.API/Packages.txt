Status 401
The 401 Unauthorized status code is a client-side error that occurs when a user tries to access a page that is 
protected by the server, but the server requires the correct login details to grant access. This status code
is similar to the 403 Forbidden status code, except that in situations resulting in this status code, 
user authentication can allow access to the resource
